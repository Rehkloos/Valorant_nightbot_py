This is a Valorant Stats API for Twitchbot (Nightbot, etc..) developed by Rehhk

This is based on Henrik-3's API scraper 

How-To:

$(urlfetch https://valorantpy.herokuapp.com/rank/(Regions: eu, ap, na, kr)/(Username)/(Tagline));

Example:

$(urlfetch https://valorantpy.herokuapp.com/rank/na/kirito/789);

**For LATAM/BR OR UNSUPPORTED COUNTRIES YOU HAVE TO USE THE "NA" region for your stats**
